# Margot Robbie - artificial girlfriend project.
This is an artifitial best girlfriend project

<img src="project-preview.JPG" alt="Image-preview">

To start whole project:
1. maven -> install
2. cd target
3. java -jar girlfriend-0.0.1-SNAPSHOT.jar

To start front-end:
1. cd front-end
2. npm install
3. npm start
